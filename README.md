# 🎨 K72 Motion Experience

A modern frontend project showcasing smooth animations and interactive UI built with **GSAP** and a stack of powerful web technologies.

---

## 🚀 Tech Stack
- **HTML5** – semantic structure
- **CSS3** – styling and layout
- **JavaScript (ES6+)** – core interactivity
- **TypeScript** – type safety and scalability
- **TailwindCSS** – utility-first styling
- **GSAP (GreenSock Animation Platform)** – advanced animations and motion effects

---

## ✨ Features
- Smooth scroll-based animations powered by GSAP
- Responsive design with TailwindCSS
- Modular and type-safe codebase using TypeScript
- Interactive UI components with reusable styles
- Optimized performance for modern browsers
